
# FastSecCheck (Proof of Concept)

**FastSecCheck** — это легковесный консольный инструмент для аудита безопасности и базового укрепления (hardening) Linux-систем.

> 📌 **Цель проекта:** Утилита написана в учебных и демонстрационных целях (тестовый концепт). Главная задача — протестировать механизмы фонового мониторинга процессов через системные вызовы, динамическую смену языков локализации, работу с `cron`-расписаниями внутри Python и реализацию интерактивного/IaC режимов автоисправления дыр. Полученные наработки будут использованы в более масштабном проекте.

## 🛠 Ключевой функционал

1.  **Контроль целостности файлов:** Проверка прав доступа на критически важные системные файлы (`/etc/passwd`, `/etc/shadow`, `/etc/group`) с возможностью их автоматического восстановления.
    
2.  **Аудит SSH:** Анализ директив конфигурации `sshd_config` (запрет авторизации по паролю и логина под `root`).
    
3.  **Управление Фаерволом:** Проверка статуса межсетевого экрана и его автоматический подъем/настройка (`ufw`) в случае отключения.
    
4.  **Поиск SUID/SGID:** Выявление потенциально опасных файлов с установленными флагами смены идентификатора пользователя/группы.
    
5.  **Мониторинг сетевой активности:** Сбор информации об открытых портах и связанных с ними процессах (дамп в `ports.log`).
    
6.  **Поведенческий анализ пользователей (Режим демона):** Отслеживание аномальной активности системных учетных записей (например, изоляция действий `www-data` вне `/var/www` и перехват выполнения запрещенных команд вроде `id` или `whoami` через подсистему `auditd`).
    

## 🚀 Особенности реализации (Архитектура теста)

-   **Многоязычность (`--Language`):** Реализована поддержка перевода вывода утилиты и логов (RU/EN) на базе встроенных словарей и модуля `locale`.
    
-   **Гибкость исправления (`-aF` / `--IaC`):** * `-aF` (Interactive) — интерактивный режим, запрашивающий подтверждение `(y/n)` у администратора перед каждым изменением конфигурации.
    
    -   `--IaC` (Infrastructure as Code) — полностью автоматический режим для интеграции в CI/CD или скрипты инициализации серверов.
        
-   **Безопасность данных (Anti-PE/LM):** Все собираемые логи и алерты изолированы в директории `/opt/FastSecCheck` с правами доступа `700 (`rwx------`)`. Обычные пользователи системы не могут прочитать отчеты, что исключает утечку данных и закрывает вектор для горизонтального перемещения (Lateral Movement) или повышения привилегий (Privilege Escalation).
    
-   **Изолированное окружение:** Установщик разворачивает Python-зависимости (модуль `croniter`) в изолированное виртуальное окружение (`venv`), не засоряя глобальные пакеты системы (согласно PEP 668).
    

## 📦 Быстрый старт

Утилита поддерживает автоматическое разрешение системных зависимостей для дистрибутивов: **Debian, Ubuntu, Arch Linux, RHEL, Fedora**.

**Установка:**

Bash

```
chmod +x install.sh
sudo ./install.sh

```

**Использование:**

Bash

```
# Вызов справки
sudo fastseccheck --help

# Интерактивная проверка с исправлением на русском языке
sudo fastseccheck -aF --Language ru

# Запуск в режиме фонового демона с проверкой каждые 5 минут
sudo fastseccheck -d "*/5 * * * *"

```

**Удаление:**

Bash

```
chmod +x uninstall.sh
sudo ./uninstall.sh

```
